PineLink: PCNHS Enrollment System
WEB-SYS1 Project | University of Baguio
A streamlined web-based enrollment platform developed for Pines City National High School (PCNHS). This project focuses on digitizing the manual registration process to enhance efficiency for both students and school administrators.

Features
Student Registration: Digital form for New, Continuing, and Transferee students.

Document Management: Secure upload system for SF9 (Report Cards) and PSA Birth Certificates.

Admin Dashboard: Centralized panel for the Registrar to approve applications and manage sections.

Real-time Status: Students can track their enrollment progress (Pending, Verified, Enrolled).

Tech Stack
Frontend: HTML5, CSS3, JavaScript (Bootstrap)

Backend: PHP

Database: MySQL

Server: Apache (XAMPP)

System Architecture
Submission: Student submits data and requirements via the web portal.

Review: Admin verifies the authenticity of digital documents.

Sectioning: System facilitates automated or manual section assignment.

Completion: Final enrollment data is stored and ready for report generation.

Academic Context
Course: WEB-SYS1 (Web Systems and Technologies 1)

Institution: University of Baguio

Objective: To demonstrate proficiency in CRUD operations, database normalization, and responsive UI design.
